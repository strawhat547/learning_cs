# vscode shortcuts

## multiple selection
```
    1、alt + click(鼠标右击)
    2、ctrl + alt + up/down 添加光标
    3、ctrl + D 选择单词
```

## fold code 
```
    1、 ctrl + shift + [ 折叠代码
    2、 ctrl + shift + ] 展开代码
```

## jump 
```
    1、 ctrl + G 跳转到某行
    2、 ctrl + T 跳转到某符号
    3、 f12 跳转到定义
    4、 shift + f12 跳转到引用出
    5、 f2 代码重构
```
